# TwitterRotate


## Installation

Add this line to your application's Gemfile:

    gem 'twitter_rotate'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install twitter_rotate

## Usage

```
Usage: tr [options]
    -v, --version      Print the version
    -h, --help         Print Help
    -t, --time         Time between prints
    -c, --count        Tweet count
    -r, --rate         Refresh Rate in seconds
```
